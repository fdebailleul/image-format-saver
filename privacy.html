Privacy Policy

Last updated: March 18, 2026

Image Format Saver does not collect, store, transmit, or share any personal data. All image processing happens locally in your browser.

1. Information We Collect

We collect no information whatsoever. Image Format Saver does not collect personal data, usage data, browsing history, or any other information about you or your activity.

2. How the Extension Works

When you convert an image, the extension fetches the image directly from its source server to your browser, converts it using the HTML5 Canvas API, and saves the result to your computer. No intermediary server is involved at any stage of this process.

3. Permissions Explained

Chrome extensions must declare the permissions they use. Here is what each permission does and why it is needed:

  • contextMenus — Adds a "Save image as…" entry to the right-click menu when you click on an image.
  • activeTab — Allows interaction with the current tab only when you explicitly trigger an action (right-click or keyboard shortcut). The extension cannot access your tabs in the background.
  • scripting — Injects a conversion script into the active tab to process the image using the Canvas API. This script runs only when you trigger a conversion.
  • downloads — Saves the converted image file to your computer's downloads folder.
  • storage — Stores your preferences (output format, quality level, resize settings) locally on your device using Chrome's Storage API. This data never leaves your device.
  • host_permissions (<all_urls>) — Required so the extension can fetch images hosted on any website for format conversion. Without this permission, cross-origin images could not be processed. Images are fetched directly from the source server to your browser.
  • notifications (optional) — Displays a brief notification when a conversion completes or fails. You can enable or disable this in the extension settings at any time.

4. Data Storage

The only data stored by the extension is your user preferences (output format, quality level, resize mode). These are saved locally on your device via the Chrome Storage API and are never transmitted anywhere.

No images are stored by the extension. Converted images are saved directly to your downloads folder and are not retained by the extension.

5. Third-Party Services

Image Format Saver does not communicate with any external server, analytics service, advertising network, or third-party API. No data of any kind leaves your browser.

6. Children's Privacy

This extension does not collect any data from any user, including children under the age of 13. Since no data is collected, no special provisions are needed under COPPA or similar regulations.

7. Changes to This Policy

If we make changes to this privacy policy, the updated version will be posted on this page with a new "Last updated" date. Significant changes will also be noted in the extension's update notes on the Chrome Web Store.

8. Contact

If you have questions or concerns about this privacy policy, you can:

  • Open an issue on the GitHub repository
  • Use the support link on the Chrome Web Store listing
